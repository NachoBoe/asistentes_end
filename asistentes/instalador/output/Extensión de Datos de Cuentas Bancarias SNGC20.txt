La funcionalidad detallada en el archivo se refiere a la extensión de datos de cuentas en el sistema Bantotal, específicamente en la plataforma SNGC20. Este archivo define una serie de campos que permiten almacenar información relacionada con cuentas bancarias, incluyendo datos numéricos, texto y fechas. Los campos están diseñados para capturar información variada, como importes, características de la cuenta y datos adicionales que pueden ser relevantes para la gestión de cuentas en el sistema.

Los campos definidos son los siguientes:

- Empresa: N (3.0)
- Cuenta: N (9.0)
- Character 1: C (1)
- Character 2: C (1)
- Texto Corto 1: C (5)
- Texto Corto 2: C (5)
- Numérico 1: N (5.0)
- Numérico 2: N (5.0)
- Numérico 1 (1 dígito): N (1.0)
- Numérico 10 (1): N (10.0)
- Numérico 10 (2): N (10.0)
- Importe 1: N (18.2)
- Importe 2: N (18.2)
- Texto Medio: C (20)
- Texto Largo 1: C (50)
- Fecha 1: D
- Fecha 2: D
- Texto Largo: V(800)

Las posibilidades que se admiten incluyen la captura y almacenamiento de información variada sobre cuentas, lo que permite una gestión más eficiente y organizada de los datos bancarios.
